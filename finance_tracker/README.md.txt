# Investment Portfolio Tracker

A Python-based application that helps users track and analyze their stock investments with real-time data, profit/loss calculations, and visual portfolio summaries. Built using `pandas`, `yfinance`, and `matplotlib`, this tool is designed for investors who want a clear, data-driven view of their holdings.

---

## ?? Features
- **Portfolio Management**: Input stocks, shares, and purchase prices from a CSV file.
- **Real-Time Data**: Fetches up-to-date stock prices via Yahoo Finance API.
- **Profit/Loss Analysis**: Calculates total returns, individual stock performance, and portfolio value.
- **Data Visualization**: Generates charts for allocation, performance, and price trends.
- **Modular Structure**: Easy to extend and customize with additional features.

---

## ??? Tech Stack
- **Python 3.9+**
- **Pandas** for data manipulation
- **Matplotlib** for visualization
- **yFinance** for stock price retrieval
