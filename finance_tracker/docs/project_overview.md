# Project Overview: Investment Portfolio Tracker

## ?? Objective
The goal of this project is to create a professional-grade, modular stock portfolio tracker that demonstrates practical software development skills while solving a real-world problem: simplifying investment tracking and analysis.

---

## ?? Key Goals
1. **Practical Use Case**: Allow retail investors to view real-time data and analyze portfolio performance.
2. **Scalable Design**: Modularize code into separate files for data fetching, calculations, and visualization.
3. **Data-Driven Insights**: Provide actionable metrics, such as portfolio diversification, profit/loss, and performance over time.
4. **Professional Presentation**: Include documentation, version control best practices, and a clean codebase.

---

## ?? Technical Highlights
- **Language:** Python 3.9+
- **Data Handling:** `pandas` for dataframes and calculations.
- **APIs:** `yfinance` for live stock market data.
- **Visualization:** `matplotlib` for portfolio charts and graphs.
- **File Structure:** Organized for easy scalability, testing, and collaboration.

---

## ??? Core Modules
| Module               | Description |
|----------------------|-------------|
| `app.py`             | Main entry point; integrates all modules. |
| `data_fetch.py`      | Handles fetching stock data from Yahoo Finance API. |
| `calculations.py`    | Contains functions for returns, P/L, and metrics. |
| `visualizations.py`  | Generates allocation and trend charts. |
| `portfolio.csv`      | Sample dataset for testing and demonstration. |
